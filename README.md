## ✨ Features

- 🧮 **Automatic bead counter** — see exactly how many beads you need per color
- 🗂️ **Three grid types** — Loom, Brick, and Peyote layouts
- 📁 **Native `.beadgrid` files** — open your patterns directly from Explorer
- 🖼️ **High-quality PNG export** — share or print your finished designs

---

## 📥 Installation

1. Go to the [**Releases**](https://github.com/rinad12/GridBead/releases/latest) page.
2. Download the latest `.exe` installer.
3. Run the installer and follow the prompts.

> Currently optimized for **Windows**. macOS and Linux support is planned.

---

## 🛠️ For Developers

**Tech stack:** Electron · React · Vite · Tailwind CSS

```bash
npm install
npm run dev
```
